Proyectos Web Realizados

Bienvenido a la sección de proyectos web. Aquí encontrarás una colección de pequeños proyectos desarrollados en html, css y javascript que reflejan nuestro crecimiento y aprendizaje. Desde aplicaciones interactivas hasta minijuegos para entretenerte, cada proyecto demuestra nuestras habilidades en desarrollo web. Explora la página y selecciona el link del proyecto que más te interese

PROYECTO WEB 1

Memorama

Este juego trata de un juego de memoria 4x4 en el que se deben encontrar los pares correctos. Si no se logran hacer coincidir los pares, el juego simplemente continúa. El juego termina cuando se encuentran todos los pares.

PROYECTO WEB 2

Alimenta A La Serpiente

Es un juego tipo consola basado en el clásico juego de la Serpiente. El usuario será representado por una serpiente recolectando cuadros, mientras la serpiente recolecte cuadros la puntuación aumenta y al mismo tiempo la cola de la serpiente. Si la serpiente choca con su cola o con uno de los bordes se termina el juego.
