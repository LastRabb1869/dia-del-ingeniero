Descripción

Este proyecto fue llevado a cabo por alumnos de SISCOM durante el segundo y tercer parcial de nuestro segundo semestre, con supervisión directa del profesor Gilberto Castro Cruz. Se elaboró una antena inalámbrica con la capacidad de conectar a nuestros dispositivos, además de poder captar el internet de la institución e integrarlo a la misma, mediante el uso de un dongle Wifi y código especifico se realizo una configuración en el lenguaje de programación Python para que todo el proceso fuera llevado a cabo de manera correcta y eficiente
