Proyectos en Consola

Proyectos Básicos Realizados

En esta sección, exploramos los cimientos de la programación a través de proyectos que se ejecutan en la terminal. Utilizando Java y Python, hemos desarrollado una serie de aplicaciones que demuestran conceptos fundamentales de la programación. Estos proyectos, aunque simples en su interfaz, son poderosos en su lógica y funcionalidad, abarcando desde algoritmos básicos hasta estructuras de datos esenciales.

PROYECTO 1

Cajero Automático Simulado Con Python

El programa “Simulador Cajero” emula las funcionalidades básicas de un cajero automático, permitiendo a los usuarios crear cuentas, iniciar sesión, consultar saldos, realizar retiros y depósitos, y pagar servicios. La clase gestiona tanto la información del usuario como el saldo del cajero. Permite; Consultar el saldo de la cuenta, consultar el saldo del cajero, realizar retiros, realizar depósitos, pagar servicios, cambiar la contraseña.

PROYECTO 2

Charly Charly

El programa “Charly Charly” Simula un ente que da respuestas a las preguntas más versátiles. El funcionamiento se basa en una interfaz interactiva en la cual el usuario puede hacer preguntas y recibir respuestas automáticas. La grabación de la respuesta comienza cuando el usuario ingresa un punto (.) y finaliza cuando se ingresa la letra z. Si no se ingresan estos caracteres, el programa continúa grabando indefinidamente.

PROYECTO 3

Calculadora De Variables Estadísticas

El programa MultiEstadistics proporciona un sistema interactivo para realizar cálculos estadísticos básicos sobre una lista de números ingresados por el usuario. Utiliza módulos de la biblioteca estándar de Python y de SciPy para calcular el promedio, varianza, desviación estándar, coeficiente de variación y curtosis de los datos.

PROYECTO 5

¿Divisible Entre 2 O No?

Este programa verifica si el número ingresado por el usuario es múltiplo de 2 o no. En caso de no ser múltiplo de 2, el programa informará al usuario. La librería utilizada es `java.util.Scanner` para la entrada de datos y tiene un tamaño de aproximadamente 498 bytes en su forma de código fuente Java proporcionado.

PROYECTO 6

Tu Número De La Suerte

Este programa te devuelve tu número de suerte basado en la fecha de nacimiento ingresada por el usuario, suma los dígitos de la fecha (día, mes, año) y luego suma los dígitos individuales de esa suma para obtener un número final. Utiliza la librería `java.util.Scanner` y tiene un tamaño de aproximadamente 641 bytes.
