Calculadora De Matrices

Sencillamente práctico

Descripción

En este programa de interfaz grafica se puede calcular la multiplicación de matrices y la matriz transpuesta ingresando valores que se le solicita al usuario.
Existen dos versiones de este programa, una realizará la multiplicación, y otro realizará la transposición. ¡Agarra la que mejor te convenga!

    • Fabricio Gonzales / José Roberto Lázaro

  • Plataforma:

    • Windows

  • Lenguaje De Desarrollo:

    • Python / Java
