CalculaDios

Cálculos sencillos a la velocidad de la luz

Descripción

Este programa es una calculadora básica que permite realizar operaciones aritméticas como suma, resta, multiplicación y división. Utiliza la librería `java.util.Scanner` para la entrada de datos del usuario y tiene un tamaño de aproximadamente 1,024 bytes en su forma de código fuente Java proporcionado.
