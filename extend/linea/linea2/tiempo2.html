Pilares de la POO

Inicios 1er cuatrimeste

Los pilares de la Programación Orientada a Objetos son las bases para el funcionamiento de la programación actual y se nos fue introducido desde el principio mediante teoría, algoritmos básicos y e investigaciones metodológicas.

Brazo y Manos hidráulicos

Inicios - Finales 1er cuatrimestre

Proyecto asignado por el profesor Gilberto Castro Cruz para la materia de física. El objetivo era poder demostrar el principio de Pascal mediante el desarrolo de un brazo y una mano hidráhulicos los cuales funcionaban controlando la presión de agua mediante jeringas.

Proyectos en Python varios

Inicios de do cuatrimeste

Con esto nos referimos a los primeros programas en python que realizamos para poder probarse en terminal y darnos una mejor idea de cómo funcionaba la programación real orientada a objetos. Dichos programas muchas veces eran simulaciones de procesos u algoritmos realizados en la vida real.

Base de datos de la uni

Inicios - Mediados 2do cuatrimeste

Proyecto asignado por el profesor Oscar David Catzim Pat para la materia de programación visual. De nuestros primeros proyectos que llegaron a requierir tanto una interfaz como también una base de datos para los alumnos registrados.

Primeras páginas web

Mediados del 2do cuatrimeste

Estas páginas no se caracterizaban principalmente por el diseño, sino por su funcionamiento, pues algunas tenían las características de trabajar con datos dados por el usuario, otras mostrar ciertas animaciones, algunas eran videojuegos web como las que presentamos y la que ves de fondo era una calculadora de promedios con base de datos en tiempo real.
